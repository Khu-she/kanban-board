

# Kanban Board

A lightweight and intuitive Kanban board application to help you visually manage your tasks and projects. This project provides a clean, browser-based interface for organizing your workflow using columns and cards.
![image](https://github.com/user-attachments/assets/7acfdddf-4596-42f0-8a1e-da4f9649ba0e)



## ✨ Features

* **🖱️ Drag and Drop Interface** Seamlessly transition tasks between columns with a smooth, intuitive drag-and-drop system.
* **✅ Comprehensive Task Management** * **Add:** Quickly spawn new tasks in any column to capture ideas instantly.  
    * **Edit:** Update task details on the fly with a single click.  
    * **Delete:** Keep your board clean by removing completed or irrelevant tasks.
* **📋 Dynamic Columns** Total control over your workflow—add or remove columns to match your specific project stages.
* **📱 Fully Responsive Design** Optimized for productivity on any device, whether you are on a desktop, tablet, or smartphone.
* **💾 Persistent Storage** Your progress is safe. The board state is saved locally, ensuring your data is right where you left it.
* **🎨 User-Friendly UI** A clean, minimalist design focused on reducing friction and making task management actually fun.

---

## 🚀 Usage

| Action | How to do it |
| :--- | :--- |
| **Creating Tasks** | Click the **"Add Task"** button in any column. |
| **Editing Tasks** | Click directly on a **Task Card** to modify details. |
| **Deleting Tasks** | Use the **Delete Option** on the specific card. |
| **Moving Tasks** | **Drag** a card and **Drop** it into a new column. |
